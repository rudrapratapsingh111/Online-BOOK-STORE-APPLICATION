# Online Book Store Application

A full-stack MERN-based web application that allows users to browse books, manage carts, and place orders.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- User authentication
- Book listing and search
- Cart and checkout functionality
- RESTful APIs for backend services

## Purpose
This project was developed as a learning-based implementation to understand full-stack web development using the MERN stack.
